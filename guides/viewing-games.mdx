## Overview

MLB.TheOhtani.com provides multiple views to help you stay on top of all MLB games. Whether you want to see today's games or plan for the week ahead, our schedule views have you covered.

## Viewing the Daily Schedule

The daily schedule shows all games for a specific date, organized by game status.

<Steps>
  <Step title="Navigate to Daily Schedule">
    Go to `/schedule/day` to see today's games, or use the **Date Picker** to select a specific date.
  </Step>
  
  <Step title="View Games by Status">
    Games are automatically grouped into three sections:
    - **Live** - Games currently in progress
    - **Preview** - Upcoming games that haven't started
    - **Final** - Completed games
    
    Each section displays the number of games in parentheses.
  </Step>
  
  <Step title="Check Game Details">
    Each game card shows:
    - Team logos and names
    - Current score (for live and final games)
    - Game time or status
    - Probable pitchers (for preview games)
    - Linescore (for live and final games)
  </Step>
</Steps>

<Tip>
Games from your favorite teams automatically appear first in each section for easier tracking.
</Tip>

## Viewing the Weekly Schedule

The weekly schedule provides a broader view of games across multiple days.

<Steps>
  <Step title="Access Weekly View">
    Navigate to `/schedule/week` or click **Weekly Schedule** from the daily view.
  </Step>
  
  <Step title="Browse by Date">
    Use the **Week Picker** to navigate between different weeks. Each day shows as an expandable section with game count.
  </Step>
  
  <Step title="Expand Date Sections">
    Click any date to expand and view all games for that day. Today's date automatically expands when you load the page.
  </Step>
  
  <Step title="Toggle All Details">
    Use the **Toggle All Details** button to expand or collapse all date sections at once.
  </Step>
</Steps>

## Understanding Game States

### Live Games

For games in progress, you'll see:
- Current score and inning
- Base runners (displayed as diamond indicators)
- Balls, strikes, and outs count
- Current batter vs pitcher matchup
- Recent plays and events

<Note>
Live games automatically refresh every 3 minutes to show the latest score and status.
</Note>

### Preview Games

Upcoming games display:
- Scheduled start time
- Probable starting pitchers
- Series information (if applicable)

### Final Games

Completed games show:
- Final score
- Complete linescore by innings
- Runs, hits, and errors totals
- Special game flags like perfect games or no-hitters

## Filtering by Sport

Use the **Sport Selector** dropdown in the header to switch between different leagues and sports (baseball, spring training, etc.).

## Season Information

At the bottom of schedule pages, you'll find:
- Current season dates
- Regular season start and end dates
- All-Star Game date
- Season progress indicator (on daily schedule during regular season)
