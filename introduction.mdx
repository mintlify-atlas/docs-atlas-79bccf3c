## Overview

MLB.TheOhtani.com is a comprehensive SaaS web application for tracking MLB games, exploring player statistics, and querying the MLB Stats API. Built with modern web technologies, it provides real-time game updates, detailed player comparisons, and an interactive API playground.

**MLB** stands for **Mitchell's Live scoreBug** - a custom scorebug experience for Major League Baseball and other leagues.

## Technology Stack

MLB.TheOhtani.com is built with:

- **SvelteKit** - Modern web framework for building fast, reactive applications
- **Tailwind CSS** - Utility-first CSS framework for responsive design
- **MLB Stats API** - Official MLB statistics and game data
- **TypeScript** - Type-safe JavaScript for robust code

<Note>
  The app features a responsive design with dark mode support, ensuring a great experience across all devices and viewing preferences.
</Note>
